# Icons

## Fetch icons from Figma

Imports icons from Figma

```
yarn import-icons
```

## Build

Minifies and creates a JS module from SVGs.

```
yarn build
```

## Config

```
{
  "figmaPersonalToken": "TOKEN",
  "fileId": "FILE_ID",
  "frame": "FRAME_NAME",
  "iconsPath": "ICON_PATH",
  "page": "PAGE"
}

```
